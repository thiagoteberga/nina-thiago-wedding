# Imagens do Site

Coloque suas 3 fotos aqui com os seguintes nomes:

- `foto-1.jpg` - Primeira foto da galeria
- `foto-2.jpg` - Segunda foto da galeria  
- `foto-3.jpg` - Terceira foto da galeria

As fotos serão automaticamente exibidas na página principal.

**Recomendações:**
- Formato: JPG ou PNG
- Tamanho recomendado: 1200x800 pixels
- Orientação: Horizontal (paisagem)

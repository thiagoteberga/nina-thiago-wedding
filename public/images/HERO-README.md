# Imagem de Fundo (Hero)

Coloque sua foto de fundo aqui com o nome:

**`hero-bg.jpg`**

## Recomendações:

### Resolução:
- **Ideal**: 1920x1080 pixels (Full HD) ou maior
- **Mínimo**: 1600x900 pixels
- **Formato**: JPG (melhor compressão) ou PNG

### Orientação:
- **Horizontal (paisagem)**
- Proporção 16:9 recomendada

### Tamanho do arquivo:
- Máximo recomendado: 500KB - 1MB
- Use ferramentas de compressão se necessário (ex: TinyPNG, ImageOptim)

### Dicas:
- ✅ Escolha uma imagem com o casal centralizado ou levemente à esquerda
- ✅ Evite imagens muito escuras (já aplicamos um filtro de escurecimento)
- ✅ Certifique-se que o texto branco ficará legível sobre a foto
- ✅ A imagem será exibida em tela cheia e responsiva

### Exemplo de posicionamento:
```
casal centralizado
      |
      v
[________________]  <- Tela completa
[    👫          ]  <- Área visível
[________________]
```

O texto "Nina & Thiago" e o countdown aparecerão sobre a imagem.
